Вступ до Trypillia

Trypillia розроблена для створення надійних систем з мінімальними витратами ресурсів. Ця сторінка допоможе вам зрозуміти базові концепції мови.

Оголошення змінних

У мові є два ключових слова для змінних: let (змінювані) та const (незмінні).

let mut_var = 10;  // Можна змінити пізніше
const PI = 3.1415; // Значення фіксоване

Функції

Функції оголошуються ключовим словом fn. Типи аргументів та результат повернення вказуються явно.

fn greet(name: String) -> String {
  return "Привіт, {name}!";
}
