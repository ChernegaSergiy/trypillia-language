TrypPM: Менеджер пакунків

Екосистема Trypillia працює на базі TrypPM. Він обробляє залежності, компілює код та допомагає публікувати ваші бібліотеки.

Популярні пакунки

  • http_server
    Високопродуктивний HTTP сервер на базі подій.
    v1.2.0
  • json_parse
    Безпечний парсер JSON без алокацій.
    v0.9.4
  • crypto_core
    Базові криптографічні алгоритми.
    v2.0.1

Шпаргалка CLI

$ tryppm install <pkg>
$ tryppm update
$ tryppm build --release
$ tryppm test
$ tryppm publish
