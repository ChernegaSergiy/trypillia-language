Специфікація мови Trypillia

Цей документ описує формальний синтаксис та семантику мови. Призначено для розробників компіляторів, аналізаторів та інструментарію.

Лексична структура

Вихідний код Trypillia складається з токенів: ключових слів, ідентифікаторів, літералів, операторів та роздільників.

  • Кодування: Усі файли повинні бути у форматі UTF-8.
  • Ідентифікатори: Починаються з літери [a-zA-Z_] і можуть містити цифри.

Граматика (EBNF)

Program = { Statement } ;
Statement = LetStatement | ReturnStatement | ExpressionStatement ;
LetStatement = "let" Identifier "=" Expression ";" ;
Expression = Term { ("+" | "-") Term } ;
